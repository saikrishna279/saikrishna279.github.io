Android Apps developed by me:

VRap Android (College Management System) App

  • VRap is a college management app made for Velagapudi Ramakrishna Siddartha Engineering College, Vijayawada.
  • Uses WebView to scrape html and Inject JS functions into the college CMS portal.
  • Authenticates students via Firebase backend.
  • Much better and convinient access to Moodle Service.
  • Advanced academic calendar to notify students of Curricular and Extra-Curricular Activities with Push Notifications.
  • Made in Java.

SLACK DoS Demo App (SlowLoris DoS)

  • A Demo app for the SlowLoris DoS Attack.
  • Slowly pings a website with a large delay between pings and doesn't let the Service close the connection.
  • Made in Java.

Turbo SMS Bomber

  • A Prank app that let's you send a load of messages to your friends and spams them.
  • Uses API Calls to multiple 3rd party servers and evades csrf token protection when necessary by using WebView instead of API Calls.
  • Made in Java.

FPS Meter app for Android

  • Shows an overlay of the screen FPS in real time.
  • Can be used as an overlay during gaming. Also shows CPU temperature.
  • Made in Java and C (Android NDK).

AFH (Android File Host) Browser

  • Let's you browse and download files from the Android File Host website.
  • Uses WebScraping and JS Injection.
  • Made in Java.
